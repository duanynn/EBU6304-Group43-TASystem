环境要求：
- JDK 8 及以上（建议 11+），已配置 JAVA_HOME 和 PATH
- Apache Maven 3.6 及以上（示例：apache-maven-3.9.13）
- Apache Tomcat 11（或兼容版本）
- 操作系统：Windows（其他系统需自行调整命令）

项目依赖：
- 所有依赖由 pom.xml 管理，使用 Maven 自动下载

编译步骤（在项目根目录执行）：
1. 打开命令行/PowerShell，进入项目根目录：
   cd "D:\大三下麦博宁\软件工程\大作业"
2. 执行 Maven 打包：
   mvn clean package
3. 打包成功后，会在 target 目录生成：
   ta-recruitment-system.war

部署到 Tomcat：
1. 将 ta-recruitment-system.war 复制到 Tomcat 的 webapps 目录
   示例：
   D:\Tomcat\Apache Software Foundation\Tomcat 11.0\webapps\
2. 启动或重启 Tomcat

访问地址（示例）：
- http://localhost:8080/ta-recruitment-system/login.jsp
